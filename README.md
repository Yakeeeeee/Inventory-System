# Inventory System with QR Feature

A modern, full-featured inventory management system built with TypeScript. This application provides a simple yet powerful solution for tracking inventory items, managing stock levels, and utilizing QR code technology for quick item identification and management.

## 📋 Overview

The Inventory System is a web-based application designed to streamline inventory management operations. It combines traditional inventory tracking with cutting-edge QR code technology to provide an efficient and user-friendly experience for businesses of all sizes.

## ✨ Features

- **QR Code Integration**: Generate and scan QR codes for quick item identification and tracking
- **Inventory Management**: Add, update, and delete inventory items with detailed information
- **Stock Tracking**: Monitor stock levels with real-time updates
- **Search & Filter**: Easily find items by various criteria
- **Responsive UI**: Works seamlessly on desktop and mobile devices
- **Data Persistence**: Secure storage of inventory data
- **User-Friendly Interface**: Intuitive design for quick learning and efficient operation

## 🛠️ Tech Stack

- **Frontend**: TypeScript (98.7%)
- **Markup**: HTML (1.3%)
- **Runtime**: Node.js

## 📦 Prerequisites

- Node.js (v14 or higher recommended)
- npm (comes with Node.js)

## 🚀 Getting Started

### Installation

1. Clone the repository:
   ```bash
   git clone https://github.com/Yakeeeeee/Inventory-System.git
   cd Inventory-System
   ```

2. Install dependencies:
   ```bash
   npm install
   ```

3. Configure environment variables (if applicable):
   - Create or update `.env.local` file with any required API keys
   - Set the `GEMINI_API_KEY` in [.env.local](.env.local) to your Gemini API key (if using AI features)

4. Run the application:
   ```bash
   npm run dev
   ```

5. Open your browser and navigate to the local server URL (typically `http://localhost:3000`)

## 📖 Usage

1. **Adding Items**: Use the interface to add new inventory items with QR codes
2. **Scanning QR Codes**: Use your device camera or QR scanner to quickly lookup items
3. **Managing Stock**: Update stock quantities and item details as needed
4. **Viewing Inventory**: Browse all items with filtering and search capabilities
